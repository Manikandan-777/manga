# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2026-04-19

### Added
- Initial release of Manga library
- 5000+ utility functions across multiple categories
- **Mathematical Operations** (112+ functions)
  - Basic arithmetic operations
  - Trigonometric functions
  - Statistical functions (mean, median, variance, std_dev)
  - Number theory (GCD, LCM, prime checking)
  - Advanced math (powers, roots, factorials, Fibonacci)
  
- **String Operations** (80+ functions)
  - Case conversion (upper, lower, camel, snake, kebab, pascal)
  - String manipulation (split, join, replace, truncate, pad)
  - String validation (email, URL, IP validation)
  - Character operations and analysis
  
- **List Operations** (75+ functions)
  - Sorting algorithms (bubble, selection, insertion, merge, quick, heap sort)
  - Searching (linear, binary search)
  - Transformation (map, filter, chunk, flatten, rotate)
  - Analysis (unique, duplicates, frequency)
  
- **Dictionary Operations** (34+ functions)
  - Key/value access and manipulation
  - Merging and filtering
  - Transformation and inversion
  
- **Type Operations** (24+ functions)
  - Type checking utilities
  - Type conversion functions
  
- **File Operations** (18+ functions)
  - File I/O (read, write, append)
  - File system operations (copy, move, delete)
  - Directory operations
  
- **Advanced Functions**
  - Data structure implementations
  - Regular expressions utilities
  - JSON operations
  - Base64 encoding/decoding
  - Date/time utilities
  - Random utilities
  
- Comprehensive documentation
  - README with quick start guide
  - Contributing guidelines
  - License (MIT)
  - Setup configuration for PyPI

### Features
- Zero external dependencies (uses only Python standard library)
- Pure Python implementation
- Well-documented with docstrings
- Organized into classes for easy navigation
- Support for Python 3.6+
