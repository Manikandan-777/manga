## Contributors

We welcome and recognize all contributions! Contributors will be listed here.

### How to be recognized as a contributor:

1. Submit a pull request with your changes
2. Have your PR approved and merged
3. You'll be added to this file automatically

### Contribution Areas

- **Code** - New functions, bug fixes, optimizations
- **Documentation** - README, docstrings, tutorials
- **Testing** - Test cases, test coverage improvements
- **Design** - Architecture improvements, refactoring
- **Maintenance** - Issue triage, release management
